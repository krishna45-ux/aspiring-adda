## ⚙️ Run and Deploy Locally

Get your development workspace running on your system with these simple commands:

### Prerequisites
*   [Node.js](https://nodejs.org/) (v18+ recommended)
*   Google Gemini API Key (for custom AI roadmaps)

### Instructions
1.  **Clone & Navigate:** Open your terminal in the root directory:
    ```bash
    cd Career_Final
    ```
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Environment Setup:**
    Create a `.env.local` file in the root directory (or duplicate `.env.example`) and configure your Gemini API Key:
    ```env
    GEMINI_API_KEY=your_gemini_api_key_here
    ```
4.  **Launch the App:** Run the local Vite server:
    ```bash
    npm run dev
    ```
    *Open `http://localhost:5173` in your browser to interact with the application!*

## 🛠️ Technology Stack & Architecture

*   **Frontend Core:** React 18, TypeScript, TailwindCSS (for modern fluid styling), and Lucide React icons.
*   **AI Engine:** Google Gemini Generative Model integrated via the state-of-the-art `@google/genai` library.
*   **Database & Auth:** Supabase integration for developer profiles, roadmap tracking, and gig bookmarks.
*   **Reflex Break Room:** HTML5 Canvas API with customized object pooling for maximum rendering performance.
*   **Server Runtime:** Express backend server equipped with Helmet and rate-limiting security suites.
